# Unsplash API를 이용한 프로젝트
[Unsplash API](https://unsplash.com/developers)

### 목적
Next.js에 익숙해지기 위해

- App Router 익숙해지기
- Data fetch 등 Next.js Document를 보고 어떤식으로 작성하는지
- React hook 익숙해지기
- 대용량 데이터를 받아올 수 있는 API이기 때문에 퍼포먼스를 고려해보기

### 개발환경
- Next.js (14.0.4)
    - React (^18)
    - Typescript

### 실행 방법
```
$ npm install
$ npm run dev
or
$ npm install
$ npm run build
$ npm run start
```

### Test
**jest**
```
$ npm run test
```
