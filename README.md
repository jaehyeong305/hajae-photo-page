# Willog 사전 코딩/과제 테스트

### 개발환경
- Next.js (14.0.4)
    - React (^18)
    - Typescript

### 실행 방법
```
$ npm install
$ npm run dev
or
$ npm install
$ npm run build
$ npm run start
```

### Test
**jest**
```
$ npm run test
```

### 주의사항
Unsplash의 API를 취득하기 위해 공개키를 발급받아 코드에 작성해 놓은 상태입니다. <br/>
기존에 공개키의 경우 하드코딩으로 작성하는 것은 보안상 좋지않은 작성법이나, 과제 제출 및 동작확인을 위해 작성해 두었습니다. (원래는 환경변수를 이용) <br/>
과제전형 결과가 발표나는 대로 해당 공개키는 폐기할 예정입니다.
